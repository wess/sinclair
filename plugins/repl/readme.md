# repl

Opens an interactive language REPL in a split to the right, so you can scratch
out an idea next to your work.

- Requires: the matching runtime — `python3`, `node`, or `bun`.
- Install: copy this folder into `~/.config/prompt/plugins/`, or add
  `plugin = /path/to/plugins/repl` to `~/.config/prompt/config`.
- Commands: "Python REPL", "Node REPL", "Bun REPL".
- Bind a key with `keybind = cmd+ctrl+r=plugin_command:repl/python`.
