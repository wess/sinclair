# lazygit

Opens [lazygit](https://github.com/jesseduffield/lazygit), the terminal UI for
git, in a new tab against the current directory.

- Requires: `lazygit` on your `PATH`.
- Install: copy this folder into `~/.config/prompt/plugins/`, or add
  `plugin = /path/to/plugins/lazygit` to `~/.config/prompt/config`.
- Keybind: `cmd+ctrl+g` (rebind with
  `keybind = cmd+ctrl+g=plugin_command:lazygit/open`).
