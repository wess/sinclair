# lazydocker

Opens [lazydocker](https://github.com/jesseduffield/lazydocker), the terminal UI
for Docker, in a new tab.

- Requires: `lazydocker` on your `PATH` and a running Docker daemon.
- Install: copy this folder into `~/.config/prompt/plugins/`, or add
  `plugin = /path/to/plugins/lazydocker` to `~/.config/prompt/config`.
- Keybind: `cmd+ctrl+d` (rebind with
  `keybind = cmd+ctrl+d=plugin_command:lazydocker/open`).
