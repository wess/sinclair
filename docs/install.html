Getting started

Install

Prompt ships from GitHub releases with a Homebrew cask for macOS and AppImage, deb, and tarball packages for Linux on both x86_64 and aarch64. You can also build it from source.

macOS

The Homebrew cask is the simplest path and keeps Prompt up to date:

brew install --cask wess/packages/prompt

Prefer a direct download? Grab the latest Prompt.dmg from the releases page, open it, and drag Prompt.app into /Applications.

Note

Release builds are signed and notarized. If macOS still warns on first launch, right-click the app and choose Open once.

Linux

Every release publishes packages for x86_64 and aarch64. Pick whichever fits your distro.

AppImage (no install)

chmod +x Prompt-*-x86_64.AppImage
./Prompt-*-x86_64.AppImage

Debian / Ubuntu (.deb)

sudo apt install ./prompt_*_amd64.deb

Tarball

tar xzf prompt-*-linux-x86_64.tar.gz
# run it directly…
./prompt-*-linux-x86_64/usr/bin/prompt
# …or install the tree under /usr/local

From source

Prompt is a Cargo workspace. The GUI lives in the app crate and the shipped binary is prompt. A recent stable Rust toolchain is required. 

git clone https://github.com/wess/prompt
cd prompt
cargo run -p app --release

To assemble distributable packages locally:

CommandProduces
scripts/bundle.shA release build assembled into dist/Prompt.app (macOS).
scripts/dmg.shdist/Prompt.dmg (run bundle.sh first).
scripts/linux.sh [x86_64|aarch64].tar.gz, .deb, and .AppImage (Linux).

First run

Launch Prompt and you get a single tab running your login shell. From here:

  • Open the in-app settings with ⌘,, or edit the config file directly — see Configuration.
  • Split the pane with ⌘D (right) or ⌘⇧D (down) — see Tabs, splits & layouts.
  • Press ⌘P to open the quick launcher and discover commands.
Tip

Config changes apply live — there's no need to restart Prompt after editing your theme, fonts, padding, or keybindings.
