Prompt documentation

Prompt is a fast, GPU-accelerated terminal for macOS and Linux. It pairs a meticulous VT core with tabs, recursive splits, live-reload config, built-in themes, a plugin system, an MCP server, and a local relay for coordinating agent teams. These docs cover all of it.
